# OBRA COLLECTION - NFT Gallery

A clean NFT gallery collection displaying algorithmic art blocks with interactive modal views and theme switching.

## Project Structure

The project has been refactored into separate HTML, CSS, and JavaScript files for better organization and maintainability:

```
website/
├── index.html          # Main HTML structure
├── styles.css          # Custom CSS styles and animations
├── data.js             # Artwork data and configuration
├── app.js              # Main JavaScript application logic
└── README.md           # Project documentation
```

## Features

- **Pixel Art Gallery**: Dynamic grid of pixel artworks with hover effects
- **Modal View**: Detailed artwork information with color palettes
- **Theme Switching**: CRT, Dark, and Light themes
- **Responsive Design**: Works on desktop and mobile devices
- **Keyboard Navigation**: Arrow keys, Enter, and Escape for navigation

## File Descriptions

### index.html
- Semantic HTML5 structure
- Clean separation of content and presentation
- External CSS and JavaScript imports
- Accessible markup with proper ARIA labels

### styles.css
- Custom pixel/CRT styling effects
- Responsive design improvements
- Custom scrollbar styling
- Print styles for accessibility

### data.js
- Centralized artwork data storage
- Easy to extend with new artworks
- Clean data structure with consistent formatting

### app.js
- Modular JavaScript architecture
- Event-driven programming
- Clear separation of concerns
- Well-documented functions
- Error handling and user feedback

## Browser Compatibility

- Modern browsers with ES6+ support
- CSS Grid and Flexbox support required
- Backdrop filter support for modal blur effects

## Development

Simply open `index.html` in a web browser to run the application. No build process or dependencies required.
